# Monopoly-Markov-Chain-Analysis

This project models the board game Monopoly as a Markov chain to compute the exact long-run probability of landing on each square under official game rules.

Although Monopoly appears to be driven purely by dice rolls, its rules introduce structural biases (jail mechanics, double streaks, card movements) 
that significantly shape player movement. By constructing an expanded state space and computing the stationary distribution, this project reveals the 
hidden probabilistic structure of the board.
